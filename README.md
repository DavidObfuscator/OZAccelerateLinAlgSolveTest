OZAccelerateLinAlgSolveTest
===========================

**A simple test for Apple Accelerate framework's system of linear equations solver**

Apple introduced Linear Algebra library into Accelerate framework (vecLib). Thatâ€™s been part of the framework since iOS 8.0 / Mac OS X 10.10.

The test solve a really simple linear equation:
```

     2y + 5z = -4
2x + 5y - 1z = 27
```
